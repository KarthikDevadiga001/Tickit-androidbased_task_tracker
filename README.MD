# Tickit – Task Tracker App

**Tickit** is a simple, design-focused Android app to help you track your daily tasks efficiently and elegantly.



## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI:** XML + Kotlin
- **Database:** Room Database
